Overview
Navigation
Page navigation is primarily done using the bottom app bar.
There may also be a back button to provide a return to the
previous view.
Cards
Most visual elements in the app are contained in cards.
Cards appear with an elevated border as marked by the
arrows below.
Some actions within the app are driven by single taps on the
relevant cards.
3-Dot Menus
You will also see 3-dot menus on some cards.
Tapping the 3-dot will drop down a menu providing actions
available on the element.
The Dashboard
Help
When you first opened the app you were presented with the
Dashboard.
To get to this page you tapped on the welcome card.
Well done, it got you here!
Record Summary
The next section shows the number of records saved and
published currently stored on your device.
Tapping either of the cards will navigate you to the relevant
view.
Theme
Next comes the theme controller.
Tapping the switch toggles the app's theme between light
and dark.
Default field values/prefixes
Tapping on a card will let you set a value that will be default
for that field.
These can be used as complete field values or prefixes in
order to make filling new record fields faster.
Records
The Records view shows records that are currently stored
on your device.
The 3-dot menu allows you to perform actions on each
record.
The floating action button allows you to create new records.
Published
The Published view shows records that have been published
as PDF files and are currently stored on your device.
The 3-dot menu allows you to perform actions on each file.
Edit/New Record
EditingOpening a new record first creates a record
populated with your default values/prefixes.
To select an image for an image field, a single tap opens the
image gallery, a double tap opens the camera.
A new record will be stored and remain as the new record
even after multiple open/close operations.
When the record is ready to be saved click the Save button.
To reset fields back to your default settings, click the Reset
button.
